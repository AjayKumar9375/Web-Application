# Ajay Kumar Portfolio

Static portfolio site for `kanekin.art`.

## Files
- `index.html`: content and structure
- `styles.css`: visual style and responsive layout
- `script.js`: footer year and reveal animation

## Run locally
```powershell
python -m http.server 5500
```
Open `http://localhost:5500`.

## Customize
1. Update email and social profile links in `index.html`.
2. Replace project cards with real project names and URLs.
3. Edit colors in `styles.css` under `:root`.
