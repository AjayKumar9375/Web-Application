Key features

01

Helm-based deployment

One umbrella release manages component dependencies and repeatable installation.

02

Production values

Persistence, resources, retention, affinity, and service settings are environment-ready.

03

Custom dashboards

Version-controlled Grafana JSON covers clusters, nodes, workloads, logs, and traces.

04

Infrastructure metrics

CPU, memory, filesystem, network, pod, node, and cluster signals are collected.

05

Actionable alerting

Alertmanager routes grouped notifications to Slack and email receivers.

06

Centralized logs

Fluent Bit enriches Kubernetes container logs before Loki storage.

07

Distributed traces

Tempo receives OTLP telemetry and connects traces to Grafana investigation.
